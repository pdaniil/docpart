Каталог в режиме 1 может работать:
- с пагинацией
- с асинхронной загрузкой товаров
Поэтому, для этого режима, запросы вынесены в общие подключаемые скрипты.

В режиме 2 и 3 - каталог может работать только с асихронной загрузкой, поэтому, для этих режимов выносить запросы в отдельный скрипт пока не нужно.
