Описание архитектуры способов доставки

В архитектуре способов доставки есть несколько основных скриптов протокола.
Основные скрипты - те скрипты, которые подключаются напрямую на соответствующих страницах, которые обеспечивают торговый процесс интернет-магазина.
Т.е. при добавлении нового способа доставки, в его протоколе обязательно наличие указанных основных скриптов.

-------------------------------------------------------------------------------------
-------------------------------------------------------------------------------------
-------------------------------------------------------------------------------------
-------------------------------------------------------------------------------------
-------------------------------------------------------------------------------------
customer_interface.php
Скрипт для вывода интерфейса покупателя при выборе способа получения.
Нужен для того, чтобы покупатель указал необходимые данные, соответствующие данному способу доставки.

-------------------------------------------------------------------------------------
show_details.php
Скрипт для отображения выбранного способа получения на странице подтверждения заказа

-------------------------------------------------------------------------------------
show_actual_info.php
Скрипт для отображения актуальной информации по доставке на странице существующего заказа.
Предназначен для вывода текущих данных по доставке, для вывода виджетов ТК, например, для отслеживания отправлений

-------------------------------------------------------------------------------------

manager_interface.php
Скрипт для интерфейса менеджера - используется в панели управления на странице карты заказа - для возможного внесения изменений и т.д.

-------------------------------------------------------------------------------------
-------------------------------------------------------------------------------------
-------------------------------------------------------------------------------------
-------------------------------------------------------------------------------------
-------------------------------------------------------------------------------------

Остальные скрипты являются специфичными для каждого способа получения и не являются основными скриптами протокола.
